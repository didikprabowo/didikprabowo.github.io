Install Postgres

2021-05-07
docker , postgres

Install use docker

  1. Create file docker compose docker-compose.yml

    version: "3"

    services:
    postgres:
        image: postgres
        ports:
        - "5432:5432"
        volumes:
        - ./dbdata:/var/lib/postgresql/data
        environment:
        - POSTGRES_PASSWORD=rahas1a
        network_mode: "bridge"

  1. Create folder config and create file postgresql.conf, copy and past file from https://github.com/postgres/postgres/blob/master/src/backend/utils/misc/postgresql.conf.sample

  2. docker-compose up

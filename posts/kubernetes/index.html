Description and tutorial

Node

Tutorial

  • Get node : kubectl get mode
  • Get detail node : kubectl describe node {node_name}

Pod

  • Small unit at a cluster.
  • Have more than one the container

Tutorial

  • Get pod : kubectl get pod | -o wide
  • Get detail pod : kubectl describe pod {pod_name}
  • Create Pod : kubectl create -f {name_file_pod.{}}
  • Expose pod : kubectl port-forward {name_pod} port_access:port_pod
  • Delete Pod : kubectl delete pod {pod_name}
  • Recreate Pod kubectl replace --force -f {filaname}
  • Access bash at Pod : kubectl exec {name_pod} --it -- bin/sh

Label

  • Penanda

Tutorial

  • Show label {syntax_prefix} --show-labels
  • Update label : kubectl label pod nginx {key}={value} --overwrite
  • Get pod by label : {syntax_prefix} -l {key}|{key}={value}

Namespaces

  • Grouping

Tutorial

  • Get Namespace : kubectl get namespace
  • Create Namespace : kubectl create -f {name_file_pod.{}}
  • Create {} with NS : kubectl create -f {name_file_pod.{}} --namespace {name_ns}
  • Delete NS : kubectl delete {name_ns}
